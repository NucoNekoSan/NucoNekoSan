# python-notes

## Overview
This repository contains personal notes and small code snippets
created while learning and practicing Python.

The purpose is to document understanding and common patterns,
not to provide a complete tutorial.

## Policy
- Keep notes short and focused
- Prefer explanation over large code blocks
- One topic per file
- Update continuously

## Contents
- Python basics and language features
- Data handling notes (pandas, built-in types)
- Small reusable code snippets
- Common pitfalls and lessons learned

## Status
- Actively updated during study
